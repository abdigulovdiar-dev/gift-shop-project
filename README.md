# Giftshop: Конструктор новогодних подарочных наборов

Веб-приложение, разработанное на Java (Spark Java) и SQLite, предназначенное для создания пользователем индивидуальных подарочных наборов. Интерфейс выполнен на HTML, CSS и JavaScript. Приложение позволяет просматривать товары в формате tier-list, формировать из них наборы, сохранять данные в базе и работать с корзиной.

## Возможности

Приложение загружает список товаров из базы данных и отображает их по категориям tier. Пользователь может сформировать собственный набор, выбрав любые позиции из каталога. После выбора товаров система рассчитывает итоговую стоимость и позволяет сохранить набор в локальной базе данных.

## Архитектура

Архитектура включает три основных уровня.

**Data Layer**. База данных SQLite, содержащая таблицы товаров и пользовательских наборов.

**Backend Layer**. Серверная часть на Spark Java, обеспечивающая маршрутизацию запросов, работу корзины, взаимодействие с базой данных и выдачу JSON-ответов.

**Frontend Layer**. Интерфейс на HTML, CSS и JavaScript, отвечающий за отображение данных, формирование наборов, работу анимаций и взаимодействие пользователя с системой.

## Категории товаров

- Категория S содержит наиболее ценные товары.
- Категория A включает популярные позиции.
- Категория B объединяет доступные товары.
- Категория C используется для недорогих элементов набора.

## Запуск проекта

Для запуска необходимо установить JDK 17 и Maven. Затем в каталоге проекта выполнить команду:

mvn exec:java -Dexec.mainClass="com.shop.App"

После запуска сервер будет доступен по адресу:
http://localhost:8085

## Использование приложения

Главная страница содержит баннер, список товаров, раздел конструктора наборов, отзывы клиентов, а также информационные блоки. Пользователь выбирает товары, задает название набора и сохраняет результат в базу данных. Корзина предоставляет возможность просматривать добавленные элементы и удалять их при необходимости.

Устойчивость и обработка данных

Приложение использует сессии Spark Java для хранения состояния корзины и базу данных SQLite для долговременного хранения пользовательских наборов. Данные передаются через REST API в формате JSON.

**Хранение информации**

Файл gifts.db содержит таблицы:

- GIFTS для хранения данных о товарах.
- CUSTOM_SETS для хранения наборов пользователей.

Каждая запись включает идентификатор, название и связанные параметры.
